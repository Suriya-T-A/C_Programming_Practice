# C Programming Practice

A collection of C programs, exercises, and problem-solving implementations created while learning C programming from scratch.

## About

This repository documents my journey of learning C programming through open-source resources, self-study, and the structured learning programs offered by Etalvis Learning. It serves as a record of my progress and a reference for future projects in embedded systems and software development.

## Objectives

* Build a strong foundation in C programming
* Improve problem-solving and logical thinking skills
* Practice writing clean and efficient code
* Understand core programming concepts through hands-on implementation
* Prepare for embedded systems and software development

## Topics Covered

* Basic Input and Output
* Data Types and Variables
* Operators and Expressions
* Conditional Statements
* Loops
* Functions
* Arrays and Strings
* Pointers
* Structures and Unions
* File Handling
* Dynamic Memory Allocation
* Practice Problems
